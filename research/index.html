Sentiment Analysis on Movie Reviews using NLP (TF-IDF)

Project Overview

In this project, a machine learning model was trained to classify movie reviews as positive or negative based on their text. By leveraging technologies such as Python, NLTK, and Scikit-Learn, the project showcases the application of Natural Language Processing (NLP) to derive meaningful insights from raw text data.

Data Ingestion

The data was sourced from the movie_reviews corpus of the NLTK library, consisting of movie reviews labeled as either 'positive' or 'negative'. It provided a comprehensive dataset for training and validating the machine learning models.

Preprocessing and Feature Extraction

Several preprocessing steps were carried out to prepare the data:

  • Tokenization: Conversion of the raw text into individual words or "tokens."
  • Stopword Removal: Removal of common words that don't carry semantic meaning.
  • Stemming: Reduction of words to their base or "stem" form.
  • Vectorization: Conversion of the processed text into numerical feature vectors using the TF-IDF method.
Model Training and Evaluation

Various machine learning models were trained and evaluated, including Logistic Regression, Random Forest, Naive Bayes, and SVM. The evaluation was conducted using the AUC-ROC metric, and the ROC curves were visualized for comparison.

Results and Conclusion

The SVM model outperformed the others, with a significant ROC_AUC score. This project has practical applications in areas like customer feedback interpretation and product recommendation. Future enhancements include experimenting with other machine learning techniques and fine-tuning the hyperparameters.

Stock Price Forecasting using SVR and Bootstrap

This project employs the Support Vector Regression (SVR) method, a machine learning algorithm used for regression problems, which builds upon the principles of Support Vector Machines (SVM). SVR employs a radial basis function (RBF) kernel, which allows the model to handle nonlinear relationships between the features and the target variable. It also uses the concept of bootstrapping, a statistical resampling technique used to estimate the variability of model predictions. In this case, 1000 bootstrap samples are generated from the training data. Each bootstrap sample is used to train the SVR model, and the model's predictions on the test set are stored. The lower and upper bounds of the prediction intervals are then calculated by taking the 2.5th and 97.5th percentiles of these bootstrap predictions. These percentiles are transformed back to the original scale of the target variable, providing a range in which we can expect 95% of the future stock prices to fall. The visualization includes the actual stock prices and the predicted stock prices along with their prediction intervals, illustrating the model's predictions and their associated uncertainty.
