Computing

I am fluent in Python, Fortran, have written a decent amount of Java, and have tinkered in a variety of other languages.

I use FLASH code to do magnetohydrodynamical simulations of white dwarf binaries. I am also writing a Deep Neural Network (DNN) for recognizing compact binary systems using data from GAIA archive.

I use Stampede2 ,the flagship supercomputer of the Extreme Science and Engineering Discovery Environment (XSEDE), for running FLASH codes and post processing which is mostly done using yt. My editor of choice is Visual Studio Code.
